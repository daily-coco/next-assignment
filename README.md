# Next-assigment

## setup

```
npx react-next-app@14
npm install @heroicons/react
npm install --save-dev @types/tailwindcss
npm i zod
npm i validator
npm i@types/validator

npm install @prisma/client@5.10.0 prisma@5.10.0
npx prisma init

// migration
npx prisma migrate dev

// Prisma Studio
npx prisma studio
```
